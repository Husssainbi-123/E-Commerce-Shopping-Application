# E-Commerce-Shopping-Application

This is a full-stack e-commerce application built with Java, JDBC, Hibernate, MySQL, HTML, CSS, and JavaScript. The application allows users to browse products, add items to the cart, place orders, and manage their account. Admins can manage products and orders through a secure panel.

Backend: Java with JDBC and Hibernate for database operations

Database: MySQL for storing user, product, and order data

Frontend: HTML, CSS, JavaScript for user interface and interaction

Features: User authentication, product CRUD, shopping cart, order processing, admin management
